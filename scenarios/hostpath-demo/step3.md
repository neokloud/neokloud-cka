# Step 3: Curl the Pod Output

Use the Node IP from Step 2:

```
curl http://<node-ip>
```

You should see output like:

```
Thank you from neokloud Sat Nov 22 12:33:47 UTC 2025
Thank you from neokloud Sat Nov 22 12:33:52 UTC 2025
Thank you from neokloud Sat Nov 22 12:33:57 UTC 2025
```

Each line is added every 5 seconds.
